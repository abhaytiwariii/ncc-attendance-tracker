<h1>🎖️ NCC Cadet Attendance System</h1>

A fully responsive web-based attendance system for NCC cadets, built with HTML, CSS, and JavaScript.
This project allows individual cadet attendance tracking, where you can add, edit, delete, and view cadet details easily.
Attendance is recorded date-wise, with support for Present/Absent marking and optional reasons for absence.

<h2>🚀 Features</h2>

✅ Responsive UI – Works on both desktop and mobile

✅ Per-cadet attendance tracking (not just whole-day)

✅ Mark Present / Absent with reasons

✅ Add, Edit, or Delete cadet details anytime

✅ Attendance records stored in localStorage (persists after refresh)

✅ Separate page to view & manage attendance history

<h2>🛠️ Tech Stack</h2>

Frontend: HTML, CSS, JavaScript

Storage: Browser LocalStorage

Hosting: GitHub Pages

<h2>📌 How to Use</h2>

Open the application in a browser.

Select a date and mark each cadet as Present/Absent.

If Absent → provide a reason.

Save attendance → records are stored in localStorage.

Open view.html to check, edit, or delete saved attendance.
